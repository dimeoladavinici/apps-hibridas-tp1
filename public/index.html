API de Alfajores Argentinos

Documentación de mi API

Autenticación JWT

Para acceder a rutas protegidas, enviá tu token en el header:

Authorization: Bearer <tu_token>

Conseguí un token:

Para acceder a rutas protegidas, necesitás un token JWT.

Para generarlo:

  1. Ejecutá el archivo generar.js con Node.js.
  2. Copiá el token que aparece en la consola.
  3. Usalo en tus peticiones como header:
    Authorization: Bearer <el_token_que_hiciste>

Ejemplos de JSON para pruebas

POST /api/fabricantes
{
  "nombre": "Guaymallén",
  "provincia": "Buenos Aires",
  "antiguedad": 60

Endpoints

/api/fabricantes
  • GET → Obtener todos
  • GET /:id → Obtener uno
  • POST → Crear nuevo
  • PUT /:id → Actualizar
  • DELETE /:id → Eliminar
  • GET ?nombre= → Buscar por nombre (puede ser sin escribirlo entero)
/api/alfajores
  • GET → Obtener todos
  • GET /:id → Obtener uno
  • POST → Crear nuevo
  • PUT /:id → Actualizar
  • DELETE /:id → Eliminar
  • GET ?relleno=&... → Filtrar por relleno
  • GET ?banado=true → Filtrar por bañado
/api/protegida
  • GET → Requiere token válido

Validaciones

Los campos requeridos deben enviarse correctamente. La API valida:

  • Tipos de datos
  • Existencia de campos obligatorios
  • Relación válida entre alfajores y fabricantes
